  • Para quem tiver curiosidade sobre o código fonte, acesse: Código
  • Para fazer o download do aplicativo, acesse: Apk Download

    • Como instalar:

    Após instalar o app, siga os seguintes passos:

  • 1. Vá em configurações
  • 2. acessibilidade
  • 3. Vai ter um serviço chamado 'Window Enumeration Service'
  • 4. Só selecionar e ativar a funcionalidade
    • Nota: As vezes tem um bug de não funcionar ao ativar pela primeira vez, então, ative e desative duas vezes se possível para garantir o funcionamento correto do serviço.
    • Para celulares Samsung, pode ser que precise de um passo a mais. Para isso, faça siga os passos 1 ao 4 e talvez o dispositivo vá te avisar que você precisa de uma permissão especial para poder ativar o serviço de acessibilidade. Para poder ativar isso, vá em: aplicativos -> WindowService (App que instalou) -> selecione os 3 pontinhos na parte de cima do celular (ou em algum outro local) e ative as permissões.
    • Feito isso, só voltar na parte de accessibilidade e ativar o serviço que vai estar funcionando.

    Qual o objetivo do projeto

  • Este projeto visa avaliar se é possível identificar unicamente usuários de celulares a partir da forma como estes utilizam um dispositivo.

    • Como funciona?

  • Os usuários instalam um aplicato que contém um serviço de acessibilidade monitorará as aplicações que o usuário utilizar. Os dados são enviados para o servidor do projeto de pesquisa, onde são processados nos experimentos científicos.

    • Quem são os responsáveis por este projeto?

  • Este projeto é uma parceria entre os pesquisadores André Grégio (UFPR) e Marcus Botacin (TAMU). Em caso de dúvidas, entre em contato com o responsável pelo projeto na sua unidade. O responśavel pelo projeto na UFPR pode ser contato pelo email

    • Sou obrigado a participar?

  • A participação nesse projeto de pesquisa é totalmente voluntária. Não participar do projeto não implica em qualquer tipo de prejuízo.

    • Minha participação é remunerada?

  • A participação nesse projeto de pesquisa é totalmente voluntária, sem qualquer tipo de compensação.

    • Quais dados são coletados?

  • Os pesquisadores coletarão apenas quais janelas o usuário seleciona durante o uso do celular (Apenas o título). Por exemplo, se você acessar o chrome, apenas o título "Google Chrome" será coletado, se acessar as configurações, apenas o título "Settings" será coletado.

    • Eu serei identificado?

  • O projeto visa identificar unicamente os usuário, mas não de-anonimizá-los. Dessa forma, seu uso do celular será identificado unicamente entre os outros usuários, mas os pesquisadores não saberão sua identidade.
